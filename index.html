Operations on file    Posted: 2014-07-21 22:07

Tip: if you see capital letter you must press SHIFT Mode legend: Normal Insert

ZZ - Save and quit    Normal
:wq - save and quit    Normal

is the same as
wq - save and exit    Command line
but to avoid mistakes I will give examples in normal mode

:q quit editing file    Normal

:q! quit without changes    Normal

:e. directory explorer    Normal

:n - edit new file    Normal

:n! - edit new file without saving current changes    Normal

:split filename - split window and load file    Normal

:sview filename - split window and load file readonly    Normal

W - next word by spaces    Normal

b - back word by punctation    Normal
B - back word by spaces    Normal
e - next word set cursor at the end word by punctation    Normal

E - next word set cursor at the end word by spaces    Normal

h - move left l - move right k - move up j - move down    Normal

:38Enter - go to 38th line    Normal

^ - first non-blank character of line    Normal
) - zero) start of line    Normal
$ - end of line    Normal
G - Go To command (prefix with number 4j moves down 4 lines.    Normal
