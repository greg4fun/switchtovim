movement

Posted: 2014-07-12 20:18


Mode legend:    Normal    Insert Tip: if you see capital letter you must press SHIFT :)

w - next word by punctation    Normal

W - next word by spaces    Normal

b - back word by punctation    Normal
B - back word by spaces    Normal
e - next word set cursor at the end word by punctation    Normal

E - next word set cursor at the end word by spaces    Normal

h - move left l - move right k - move up j - move down    Normal

:38Enter - go to 38th line    Normal

^ - first non-blank character of line    Normal
) - zero) start of line    Normal
$ - end of line    Normal
G - Go To command (prefix with number 4j moves down 4 lines.    Normal

Windows:

Ctrlww - swith between windows    Normal

Ctrl w ⇧ - move cursor up a window example using ctr ww and ctr w arrows in example before switching windows ESC was pressed to enter normal mode:

ctrl w+ - increase size of current window    Normal

ctrlw- - decrease size of current window    Normal

ctrlw_ - maximize current window    Normal

ctrlw= - make all equal size    Normal

10ctrlw+ - increase window size by 10 lines    Normal
vsplit file - vertical split    Command line
sview - same as split, but readonly    Command line
hide - close current window    Command line
only - keep only this window open    Command line

ls - show current buffers    Command line
b 2 - open buffer #2 in this window    Command line
ctrlWs - split window horizontally    Normal

ctrlWv - split window vertically    Normal

ctrlwq - close window    Normal
