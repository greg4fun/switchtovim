Operations on file

Posted: 2014-07-21 22:07

Tip: if you see capital letter you must press SHIFT Mode legend: Normal Insert

ZZ - Save and quit    Normal
:wq - save and quit    Normal

is the same as
wq - save and exit    Command line
but to avoid mistakes I will give examples in normal mode

:q quit editing file    Normal

:q! quit without changes    Normal

:e. directory explorer    Normal

:n - edit new file    Normal

:n! - edit new file without saving current changes    Normal

:split filename - split window and load file    Normal

:sview filename - split window and load file readonly    Normal
