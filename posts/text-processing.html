text processing

Posted: 2014-07-21 23:10

Tip: if you see capital letter you must press SHIFT Mode legend: Normal Insert

i - enter insert mode and place cursor at current position    Insert
I - enter insert mode and place cursor at start of the line    Normal
a - enter insert mode and place cursor after current position    Normal
A - enter insert mode and place cursor at the end of line    Normal
ea - enter insert mode and place cursor at the end of line    Normal
J - join line below with current one    Normal
cc - change replace entire line S - change replace entire line    Normal
cw - change replace to the end of word    Normal
c$ - change replace to the end of line    Normal
d$ - delete after cursor D - delete after cursor    Normal
u - undo    Normal
ctrlr - redo    Normal
. repeat last command    Normal
